🛠️ Guia de Montagem: Organizador de Engrenagens
Siga os passos abaixo com atenção à orientação das peças para garantir o funcionamento suave do mecanismo.

Fase 1: Mecanismo Central
Prepare a Base: Posicione a peça da base em uma superfície plana.

Eixo Principal: Encaixe a engrenagem grande com eixo exatamente no centro da base.

Engrenagens de Tração: Localize as engrenagens com furo quadrado. Encaixe-as de modo que a face plana do encaixe esteja direcionada para o eixo central.

Engrenagens Complementares: Encaixe as demais engrenagens (sem o encaixe quadrado), mantendo a mesma orientação das anteriores para garantir o alinhamento dos dentes.

Fase 2: Estrutura e Compartimentos
Trava Inferior: Posicione a tampa com os encaixes laterais por cima das engrenagens, pressionando levemente até firmar.

Organização das Gavetas: Insira as gavetas observando a curvatura lateral, que deve acompanhar o desenho da base. Siga rigorosamente esta ordem de valores:

10 → 100 → 220 → 1k → 10k

Fechamento Superior: Encaixe a tampa decorada (com desenho) sobre o conjunto de gavetas.

Fase 3: Finalização
Trava de abertura: Finalize o conjunto rosqueando a peça em formato de estrela no topo do eixo central. Ela serve para abrir as gavetas, portanto se tiver abrindo deu tudo certo!
